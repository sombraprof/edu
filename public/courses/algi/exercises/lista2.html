Lista 2 — Condicionais e operadores

Aplique estruturas de decisão (if, else if, switch) e operadores lógicos para resolver os desafios.

  1. Faça um programa que leia idade e categoria de um atleta e classifique-o nas modalidades infantil, juvenil ou adulto.
  2. Leia um número inteiro e informe se ele é múltiplo de 3, de 5, de ambos ou de nenhum.
  3. Construa um menu para calcular a área de figuras planas (quadrado, retângulo, triângulo e círculo) a partir de dados fornecidos pelo usuário.
  4. Implemente um algoritmo que avalie uma expressão booleana informando quando ela for verdadeira ou falsa com base em entradas do usuário.

Organize o código em funções para incentivar a reutilização de trechos em exercícios futuros.
