Exercício — Sincronização com API

Crie um aplicativo simples com listagem de itens e suporte a modo offline.

  • Consuma dados de uma API pública (ou mock) utilizando HTTP client.
  • Armazene o resultado localmente (SQLite/Room/Core Data/IndexedDB).
  • Atualize o cache ao voltar para o modo online, exibindo feedback para o usuário.
  • Teste interrupções de rede e conflitos de dados, registrando as decisões tomadas.
