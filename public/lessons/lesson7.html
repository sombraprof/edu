O Papel dos Operadores

Se pensarmos no fluxo de um programa como Entrada → Processamento → Saída, os operadores são o coração do Processamento. São eles que nos permitem pegar os dados de entrada, transformá-los, combiná-los e tomar decisões para, no fim, gerar uma saída útil.

Dominar os operadores é o primeiro passo para dar "inteligência" e funcionalidade aos nossos programas.

Operadores Aritméticos

São os símbolos que usamos para realizar as operações matemáticas básicas.

Operador Descrição Exemplo
+ Adição 5 + 3 // resulta em 8
- Subtração 5 - 3 // resulta em 2
* Multiplicação 5 * 3 // resulta em 15
/ Divisão 10 / 2 // resulta em 5
% Módulo (resto da divisão inteira) 10 % 3 // resulta em 1
Cuidado com a Divisão de Inteiros!

Em C, a divisão de dois números inteiros resulta sempre num número inteiro (a parte decimal é truncada). Por exemplo, 5 / 2 resulta em 2, e não 2.5. Para obter o resultado decimal, pelo menos um dos números deve ser do tipo float ou double: 5.0 / 2 resulta em 2.5.

Operadores Relacionais

São usados para comparar dois valores. O resultado de uma comparação é sempre um valor booleano: verdadeiro (representado por 1 em C) ou falso (representado por 0).

Operador Descrição Exemplo
== Igual a 5 == 5 // resulta em 1 (verdadeiro)
!= Diferente de 5 != 3 // resulta em 1 (verdadeiro)
> Maior que 5 > 3 // resulta em 1 (verdadeiro)
< Menor que 5 < 3 // resulta em 0 (falso)
>= Maior ou igual a 5 >= 5 // resulta em 1 (verdadeiro)
<= Menor ou igual a 5 <= 3 // resulta em 0 (falso)

Operadores Lógicos

Permitem-nos combinar múltiplas condições para criar decisões mais complexas.

Operador Nome Lógico Exemplo
&& E (AND) (idade >= 18 && temCarta == 1)
|| OU (OR) (dia == 6 || dia == 7) // Sábado ou Domingo
! NÃO (NOT) !(idade >= 18) // Verdadeiro se a idade NÃO for maior ou igual a 18

Precedência de Operadores

Quando temos uma expressão complexa como 5 + 3 * 2, qual operação é feita primeiro? A linguagem C segue uma ordem de precedência, muito similar à da matemática.

  1. Parênteses () (maior prioridade)
  2. Operadores Aritméticos de Multiplicação e Divisão (* / %)
  3. Operadores Aritméticos de Adição e Subtração (+ -)
  4. Operadores Relacionais (> < == != etc.)
  5. Operadores Lógicos (&& e depois ||) (menor prioridade)
Boa Prática: Use Parênteses!

Mesmo que saiba a ordem de precedência, use parênteses para agrupar as suas expressões. Isto torna o código muito mais fácil de ler e entender, tanto para si como para outros programadores, e evita erros lógicos difíceis de encontrar.

Atividade Prática em Duplas

Em duplas, vamos criar um programa que calcula a média ponderada de três notas. A fórmula é: Media = (Nota1 * Peso1 + Nota2 * Peso2 + Nota3 * Peso3) / (Peso1 + Peso2 + Peso3).

  1. Declarem variáveis float para as 3 notas e 3 pesos.
  2. Peçam ao utilizador para inserir as notas e os pesos.
  3. Calculem a média ponderada numa variável float chamada mediaPonderada. Usem parênteses para garantir a ordem correta dos cálculos!
  4. Exibam o resultado de forma clara, mostrando as notas, os pesos e a média final.

Próximos Passos e Tarefa (TED)

Objetivo da Tarefa

Consolidar o entendimento sobre operadores aritméticos, relacionais e lógicos em C, praticando precedência de operações e a formatação detalhada da saída no console. Esta atividade reforça a importância de escrever código legível e explicativo.

Instruções

  1. Elabore um programa em C que calcule a média ponderada de três notas com pesos diferentes. Você pode reutilizar os valores usados em aula ou criar novos.
  2. A saída deve ser autoexplicativa, mostrando o cálculo completo no printf, por exemplo:
    (Nota1 * Peso1 + Nota2 * Peso2 + Nota3 * Peso3) / (Peso1 + Peso2 + Peso3) = Resultado
  3. Utilize parênteses para deixar clara a ordem das operações e evitar ambiguidades.
  4. Comente o código explicando o papel de cada variável e a lógica aplicada. Isso ajudará a fixar os conceitos de forma mais profunda.
  5. Salve o arquivo como aula7_media_ponderada.c e poste o código na Atividade "Aula 7" no Moodle.
Dica Extra

Teste seu programa com diferentes combinações de notas e pesos, incluindo valores fracionários (ex.: 8.5) e pesos variados. Isso garantirá que seu código funciona para diversos cenários e que você entendeu bem o uso de float e double.
