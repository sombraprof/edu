Quando o if-else não é suficiente?

Na última aula, vimos como o else if nos permite criar múltiplos caminhos. No entanto, quando todas as comparações são feitas contra a **mesma variável**, o código pode ficar repetitivo e menos legível.

Imagine um menu onde o utilizador digita um número para escolher uma opção. Usar if (opcao == 1), else if (opcao == 2), else if (opcao == 3)... funciona, mas não é elegante.

O switch-case é uma estrutura de controlo desenhada exatamente para esta situação: selecionar um bloco de código para executar com base no valor de uma única variável ou expressão.

Anatomia do switch-case

A estrutura switch avalia uma variável e "salta" para o case que corresponde ao valor encontrado. 

switch (variavel_ou_expressao) {
    case valor1:
        // Bloco de código para o valor1
        break;
    case valor2:
        // Bloco de código para o valor2
        break;
    // ... outros cases
    default:
        // Bloco de código se nenhum case corresponder
        break;
}
  • switch (variavel): A variável (do tipo int ou char) que será avaliada.
  • case valor:: Um rótulo que representa um possível valor para a variável. O valor deve ser uma constante.
  • break;: **Essencial!** Interrompe a execução do switch. Sem ele, o código continuaria a executar os cases seguintes (comportamento "fall-through").
  • default:: Opcional. Funciona como o else final, executando se nenhum dos cases anteriores corresponder ao valor da variável.

Exemplo Prático 1: Menu de Opções

Vamos implementar um menu de opções simples, um dos usos mais comuns para o switch-case. 

#include <stdio.h>

int main() {
    int opcao;

    printf("--- MENU ---\n");
    printf("1 - Cadastrar\n");
    printf("2 - Listar\n");
    printf("3 - Sair\n");
    printf("Escolha uma opcao: ");
    scanf("%d", &opcao);

    switch (opcao) {
        case 1:
            printf("Voce escolheu a opcao CADASTRAR.\n");
            // Aqui entraria a lógica de cadastro
            break;
        case 2:
            printf("Voce escolheu a opcao LISTAR.\n");
            // Aqui entraria a lógica de listagem
            break;
        case 3:
            printf("A sair do programa...\n");
            break;
        default:
            printf("Opcao invalida!\n");
            break;
    }

    return 0;
}

Agrupando cases

Uma técnica poderosa é agrupar vários cases para que executem o mesmo bloco de código. Como a execução continua até encontrar um break, podemos omiti-lo para criar este efeito.

Exemplo: Dias do Mês

Este programa lê o número de um mês e informa quantos dias ele tem (ignorando anos bissextos). 

#include <stdio.h>

int main() {
    int mes;
    printf("Digite o numero do mes (1-12): ");
    scanf("%d", &mes);

    switch (mes) {
        // Meses com 31 dias
        case 1:
        case 3:
        case 5:
        case 7:
        case 8:
        case 10:
        case 12:
            printf("Este mes tem 31 dias.\n");
            break;

        // Meses com 30 dias
        case 4:
        case 6:
        case 9:
        case 11:
            printf("Este mes tem 30 dias.\n");
            break;

        // Caso especial: Fevereiro
        case 2:
            printf("Este mes tem 28 ou 29 dias.\n");
            break;

        default:
            printf("Mes invalido!\n");
            break;
    }

    return 0;
}

Atividade Prática em Duplas

Em duplas, criem um programa que leia um único caractere representando o estado civil de uma pessoa (S para Solteiro, C para Casado, V para Viúvo, D para Divorciado) e imprima o estado civil por extenso.

Dica

O switch também funciona com o tipo char. Lembre-se de que os valores dos cases para caracteres devem estar entre aspas simples, como case 'S':.
