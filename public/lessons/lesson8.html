Fluxo Sequencial e Múltiplas Leituras com scanf

Até agora, focámos em ler um ou dois valores. No mundo real, os programas precisam de lidar com múltiplas entradas de dados. A função scanf é poderosa e permite-nos ler vários valores de uma só vez, consolidando a etapa de **Entrada** do nosso fluxo. 

// Read two prices in a single line
float p1, p2;
printf("Digite o preço de dois produtos (ex: 10.50 8.75): ");
scanf("%f %f", &p1, &p2); // Make sure to pass addresses (&)
Atenção com scanf!
  • O "Enter" Fantasma: Após ler um número, a tecla "Enter" (\n) que você pressiona fica no "buffer" de entrada. Se a próxima leitura for de um caractere (%c), ela pode ler esse "Enter" em vez do que você quer. Uma forma de resolver é adicionar um espaço antes do %c: scanf(" %c", &variavel);.
  • Tipos Corretos: Usar %d para ler um float (ou vice-versa) pode causar erros imprevisíveis. Verifique sempre se o especificador de formato corresponde ao tipo da variável.
  • O & é Obrigatório: Esquecer o & antes do nome da variável em scanf é um dos erros mais comuns. Lembre-se que scanf precisa saber o **endereço** da "caixa" onde vai guardar o valor.

Cálculos Encadeados e Variáveis Intermediárias

Para problemas mais complexos, raramente chegamos ao resultado final num único passo. A etapa de **Processamento** muitas vezes envolve cálculos sequenciais, onde o resultado de uma operação serve de entrada para a próxima. Usar **variáveis intermediárias** para guardar estes resultados torna o código mais legível e fácil de depurar.

Exemplo: Descontos Sucessivos

Um desconto de 10% seguido por um de 5% não é o mesmo que um desconto de 15%. A ordem importa, e o segundo desconto aplica-se sobre o novo valor. 

// Apply 10% then 5% sequential discounts
float price = 100.0f;
float afterFirst, finalPrice;

afterFirst = price * 0.90f;          // First 10% discount (price becomes 90.00)
finalPrice = afterFirst * 0.95f;     // Then 5% discount over the new base (90.00 * 0.95)

printf("Final price: R$ %.2f\n", finalPrice); // Prints R$ 85.50

Saída Formatada com printf

A etapa de **Saída** é a forma como o nosso programa comunica os resultados ao utilizador. Uma saída bem formatada é crucial para que a informação seja clara e profissional.

Boas Práticas de printf
  • Controle de Decimais: Use %.2f para formatar valores monetários ou outros números que precisem de exatamente duas casas decimais.
  • Alinhamento: Pode controlar a largura de um campo. Por exemplo, %8.2f reserva 8 espaços para o número, alinhando-o à direita, o que é ótimo para criar tabelas e recibos.
  • Caracteres de Escape: Use \n para quebrar uma linha e \t para adicionar uma tabulação (um espaçamento padronizado).

Atividade Prática em Duplas: Recibo de Compras

Em duplas, vamos criar um programa que simula um recibo de compras. O programa deve ler o preço e a quantidade de 3 produtos, calcular o subtotal de cada um, o total geral, aplicar um desconto de 10% e exibir um recibo bem formatado. 

#include <stdio.h>

int main() {
    // --- INPUT VARIABLES ---
    float price1, price2, price3;
    int qty1, qty2, qty3;

    // --- PROCESSING VARIABLES ---
    float subtotal1, subtotal2, subtotal3;
    float total, discount, finalTotal;

    // --- INPUT STAGE ---
    printf("Digite o preco e a quantidade do produto 1: ");
    scanf("%f %d", &price1, &qty1);

    printf("Digite o preco e a quantidade do produto 2: ");
    scanf("%f %d", &price2, &qty2);
    
    printf("Digite o preco e a quantidade do produto 3: ");
    scanf("%f %d", &price3, &qty3);

    // --- PROCESSING STAGE ---
    subtotal1 = price1 * qty1;
    subtotal2 = price2 * qty2;
    subtotal3 = price3 * qty3;
    total = subtotal1 + subtotal2 + subtotal3;
    discount = total * 0.10f; // 10% discount
    finalTotal = total - discount;

    // --- OUTPUT STAGE ---
    printf("\n\n--- RECIBO DE COMPRAS ---\n");
    printf("Produto 1: %d x R$ %.2f = R$ %8.2f\n", qty1, price1, subtotal1);
    printf("Produto 2: %d x R$ %.2f = R$ %8.2f\n", qty2, price2, subtotal2);
    printf("Produto 3: %d x R$ %.2f = R$ %8.2f\n", qty3, price3, subtotal3);
    printf("----------------------------------\n");
    printf("Total Bruto:              R$ %8.2f\n", total);
    printf("Desconto (10%%):           R$ %8.2f\n", discount);
    printf("Total a Pagar:            R$ %8.2f\n", finalTotal);
    printf("----------------------------------\n");

    return 0;
}
